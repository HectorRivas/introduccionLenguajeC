Introduccion al lenguaje

Que es

El lenguaje C es un lenguaje de programacion estructurado, en el mismo sentido que lo son otros lenguajes de programación tales como el lenguaje Pascal, el Ada o el Modula-2, pero no es estructurado por bloques, o sea, no es posible declarar subrutinas (pequeños trozos de programa) dentro de otras subrutinas, a diferencia de como sucede con otros lenguajes estructurados tales como el Pascal.

Este lenguaje fue creado por Brian Kernighan y Dennis Ritchie a mediados de los años 70. La primera implementación del mismo la realizó Dennis Ritchie sobre un computador DEC PDP-11 con sistema operativo UNIX. C es el resultado de un proceso de desarrollo que comenzó con un lenguaje anterior, el BCPL, el cual influyó en el desarrollo por parte de Ken Thompson de un lenguaje llamado B, el cual es el antecedente directo del lenguaje C. El lenguaje C es un lenguaje para programadores en el sentido de que proporciona una gran flexibilidad de programación y una muy baja comprobación de incorrecciones, de forma que el lenguaje deja bajo la responsabilidad del programador acciones que otros lenguajes realizan por si mismos. Así, por ejemplo, C no comprueba que el índice de referencia de un vector (llamado array en la literatura informática) no sobrepase el tamaño del mismo; que no se escriba en zonas de memoria que no pertenecen al área de datos del programa, etc.

Desde su creación, surgieron distintas versiones de C, que incluían unas u otras características, palabras reservadas, etc. Este hecho provoco la necesidad de unificar el lenguaje C, y es por ello que surgió un standard de C, llamado ANSI-C, que declara una serie de características, etc., que debe cumplir todo lenguaje C.

El standard ANSI-C contiene un total de 32 palabras reservadas, las cuales son:

auto break case char
const continue default do
double else enum extern
float for goto if
int long register return
short signed sizeof static
struct switch typedef union
unsigned void volatile while

Este lenguaje es compilado por lo que se requiere de un programa que nos ayude con esta tarea para que podamos compilar nuestro codigo y ejecutarlo por consola. Existen diversos editores de texto que nos ayudan a realizar esta tarea. Los mas populares son Code-Blocks y Dev-c++ mismos que puedes descargar e instalar de forma gratuita desde su pagina de intenert.

Primer programa "Hola Mundo"

Si ya cuentas con un IDE instalado ya puedes hacer tu primer programa en C. Para ello solo necesitas conocer las estructura de un programa basico. Observa el siguiente codigo que nos muestra un mensaje por consola de "Hola mundo".

#include < stdio . h>

main()
{
   printf("Hola mundo");
   return 0;
}

Listo! Ya tienes tu primer programa hecho en lenguaje C

Identificadores, tipos de datos y variables

Indentificadores

Un indentificador es aquel nombre dado por el programador para declarar una variable y poder hacer uso de ella dentro de nuestro codigo. Recordando que C es un lenguaje sensible al contexto por lo que distingue entre mayusculas y minusculas por lo que la misma palabra podria ser tomada como dos Indentificadores diferentes, por ejemplo: "nombre" y "NOMBRE" son dos Indentificadores.

Las formas mas comunes de crear indentificadores son:

nombre_variable (Separando dos palabras con un guion bajo)

nombreVariable (Separando dos palabras empezando la segunda con mayuscula)

nombreVariable2 (Mismo nombre pero con numero)

Tipos de datos

Al igual que en otros lenguajes de programacion, C tambien cuenta con diferentes tipos de datos que son muy importantes ya que de estos depende que hagamos un correcto uso de las variables y desarrollo de codigo. Los tipos de datos son los que se muestran a continuacion:

int Enteros
float numeros reales
char caracter
char[ ] cadena de caracteres
void sin valor (uso especial)

Variables

La sintaxis correcta para poder declarar una variable es la siguiente:

tipo_de_dato nombre_variable;

Ejemplo

int a;

Con esto ya esta declarada una variable de tipo entero que resive como nombre "a".

Incluso se le puede dar un valor inicial y nos quedaria de la siguiente manera:

int a=0;

De esta manera tambien declaras la variable solo que le has dado un valor de inicio el cual puede cambiar o no dependiendo la utilizacion que se le de dento del codigo.

Codigo que imprime el valor de una variable

#include < stdio . h>

int a=5;

main()
{
   printf("El valor de 'a' es: %d", a);
   return 0;
}
